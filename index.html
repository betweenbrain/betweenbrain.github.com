What the funk?

Hello. This is Jekyll Me, a thing made by Seth Warburton, using the awesome Jekyll.

The intention is to provide you with a solid basis on which to build your own awesome Jekyll-powered blog running, for free on Github. It's the new hotness, dontcha know.

All the latest buzzwords are built right in; responsive, mobile-first, content first, html5, css3, LESS. I left out all the old boring stuff.

Easy, peasy, lemon, squeezy ®

If you can figure out how to create a Github account, have some idea what you are doing with html and css and know all about copying/pasting you should be fine.

You don't really need to install anything, but for lack of a better word let's call it and install process. It goes like this:

  1. If you don't have one, get a Github account.
  2. Fork this repository: https://github.com/nternetinspired/nternetinspired.github.com
  3. Click the 'settings' button and rename the repository to your-github-username.github.com
  4. Wait a couple of minutes, then visit your-github-username.github.com in your browser. Presto.
